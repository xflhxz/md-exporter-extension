# 隐私政策 | Privacy Policy

**最后更新 | Last Updated: 2024-12**

## 中文

### 数据收集

**Page to Markdown 扩展不会收集任何个人数据或浏览信息。**

- **不收集** — 本扩展不会收集、存储或传输您的任何个人信息、浏览历史、页面内容或其他数据。
- **不上传** — 所有操作均在您的浏览器本地完成。页面内容不会被发送到任何远程服务器。
- **不追踪** — 本扩展不包含任何分析、统计或追踪代码。
- **无第三方** — 本扩展不集成任何第三方服务或 SDK。

### 权限说明

本扩展请求的 Chrome 权限仅用于实现核心功能：

| 权限 | 用途 |
|------|------|
| `activeTab` | 访问当前标签页以提取页面内容 |
| `scripting` | 注入内容脚本来执行 Markdown 转换 |
| `storage` | 保存您的导出选项（如是否保留图片） |
| `contextMenus` | 添加右键菜单项，方便快速导出 |

### 数据存储

本扩展仅在您的本地浏览器中使用 `chrome.storage.local` 存储您的偏好设置（YAML 元数据、图片、链接等选项的开关状态）。这些数据不会离开您的设备。

### 联系我们

如有隐私相关问题，请通过 GitHub Issues 联系我们。

## English

### Data Collection

**Page to Markdown extension does NOT collect any personal data or browsing information.**

- **No Collection** — This extension does not collect, store, or transmit any personal information, browsing history, page content, or other data.
- **No Upload** — All operations are performed locally in your browser. Page content is never sent to any remote server.
- **No Tracking** — This extension contains no analytics, statistics, or tracking code.
- **No Third Parties** — This extension does not integrate any third-party services or SDKs.

### Permission Justification

The Chrome permissions requested by this extension are used solely for core functionality:

| Permission | Purpose |
|------------|---------|
| `activeTab` | Access the current tab to extract page content |
| `scripting` | Inject content script to perform Markdown conversion |
| `storage` | Save your export preferences (e.g., whether to include images) |
| `contextMenus` | Add right-click menu items for quick export |

### Data Storage

This extension only uses `chrome.storage.local` to store your preference settings (toggle states for YAML metadata, images, links, etc.) locally in your browser. This data never leaves your device.

### Contact

For privacy-related concerns, please reach out via GitHub Issues.
